Digital Art

Processing 3 (Java), Photoshop

In Spring 2018, I enrolled in ARTT370: Elements of Digital. I programmed using Java in Processing to draw and render graphical components on the canvas, as well as utilize concepts of generative art.

Project 1: For this project, I utilized basic means of creating imagery in Processing, such as points, lines, curves, and shape primitives.

Project 2: Here, I used variables and an iterative routine to create imagery


Project 3: This project examines the notion of "emergent systems" in which the rules and actors within the system coalesces to create something greater than its parts. Here, I made use of classes, arrays and array lists to create the firework and explosion effects.

Project 4: The Yahoo weather API data for College Park serves as alternative input (other than user input, i.e. mouse and keyboard) to generate imagery. I used a variety of images edited with Photoshop and animated them together.

Traditional Art

Graphite and charcoal

In Spring 2017, I enrolled in my first art class, ARTT110: Elements of Drawing I (Art below is in chronological order). I learned how to use different graphite pencils to different effects, about perspective, reflections and shading, as well as different techniques/materials to draw contours and portray still life/realistic figures.
